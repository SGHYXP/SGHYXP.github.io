GitHub Pages搭建博客

  • sghyxp
  • 2018-10-28
  • GitHub Pages
  • 博客搭建

1.GitHub

GitHub是一个共享虚拟主机服务，用于存放使用Git版本控制的软件代码和内容项目(引自维基百科）

2.GitHub Pages

github有一个很有爱的项目，叫做github pages，这个项目是给开发者建立一个私人页面，上面用来分享新颖的想法和自己写的代码.

3.注册自己的GitHub账号

首先进入Github站点,然后进行注册(此处不做详细说明可自行阅读github教程：[1]注册github)

4.创建仓库

点击右上角头像左边的➕号按钮,选择New repository新建仓库,如图:


5.设置GitHub Pages

进入设置页面


如果仓库名是以:用户名.github.io命名, 仓库创建成功后默认开启Github Pages.
找到下面选择主题的地方:

选择一个主题,做为GitHub Pages的主题,然后会跳到README.md文件,可以预览博客页面效果:

不做修改可以直接点击下面的Commit changes提交.
就可以通过用户名.github.io访问自己的博客了.比如:sghyxp的博客
新建的另一个非用户名命名的仓库:1234访问结果是404.

搭建完成的GitHub Pages博客,如:sghyp
